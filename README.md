## Live Pages

### 🏢 Enterprise GTM Engine
**[falconontop.github.io/enterprise-gtm-engine](https://falconontop.github.io/enterprise-gtm-engine)**

Account-based pipeline architecture for complex B2B sales. Built for GTM motions where the deal is multi-threaded, the buyer is a committee, and volume outbound would embarrass you.

- 7-layer architecture: account universe → buying committee mapping → signal layer → multi-score qualification → multi-thread outreach → CRM output → feedback loop
- Three-score qualification gate (Fit, Intent, Timing) with positive + negative weights and hard disqualifiers
- AI-powered agents (Claude) for account research, org mapping, signal briefs, role-specific sequence generation, and feedback ingestion
- Python pipeline — Claude subprocess pattern, structured JSON contracts between layers
- CRM-agnostic output design

### ⚡ SMB GTM Engine
**[falconontop.github.io/signal-based-gtm-engine](https://falconontop.github.io/signal-based-gtm-engine)**

Signal-based outbound infrastructure for B2B SaaS. Replaces manual research with automated signal detection, dual-score qualification, and AI-assisted sequence generation.
